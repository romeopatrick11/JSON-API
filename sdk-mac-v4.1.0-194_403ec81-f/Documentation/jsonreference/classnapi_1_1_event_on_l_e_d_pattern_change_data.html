Detailed Description

Notify the NEA of the currently known provisioning LED patterns.

The patterns will be in a JSON array. There will be no duplicates. The array may be empty when a pattern is no long available (for example, when the Nymi Band with the pattern leaves provisioning mode).

The patterns are as described for napi::ProvisionPattern.
