Detailed Description

Response to the Roaming Authentication Sig Request.

This response to the Roaming Authentication signature request contains two results:

  1. the raKeyId provided by the Nymi Band upon successful completion of the RoamingAuthSetupResp
  2. the Nymi Band signature of the server nonce.

At this point the Nymi Band is convinced that the Roaming Authentication service is valid. However, the Roaming Authentication service needs the Nymi Band signature before it is convinced that the Nymi Band is valid.

The Nymi Band signature should be forwarded to the Roaming Authentication service. If the service is satisfied it will normally respond with the additional information about the user. This user information is based, at least in part, on the user information provided to it during the Roaming Authentication setup process.
