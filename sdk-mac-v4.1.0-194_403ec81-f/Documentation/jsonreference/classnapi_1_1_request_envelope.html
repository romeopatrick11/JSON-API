Detailed Description

The envelope wrapping the NAPI JSON request.

When a request is made to NAPI it is wrapped in this JSON object. The response from NAPI is wrapped in the same object with some additional attributes defined.

The exchange attribute is an arbitrary string determined by the NEA that can be used to match this request to its response.
